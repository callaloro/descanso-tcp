# Descanso mínimo obligatorio en vuelo TCP - PWA

Archivos para publicar como app instalable offline.

Para que se pueda instalar y funcionar offline en Android/iPhone, debe abrirse desde HTTPS, por ejemplo GitHub Pages, Netlify o Vercel.

Una vez publicada:
1. Abrir el enlace en Chrome/Safari.
2. Menú del navegador.
3. Añadir a pantalla de inicio / Instalar app.
4. Abrirla una vez con internet para que guarde los archivos.
5. Después funcionará sin conexión.
